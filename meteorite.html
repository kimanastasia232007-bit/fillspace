1. What is a REST API?

A REST API is a web service that allows programs to communicate over HTTP/HTTPS.

  • Client sends a request to a URL.
  • Server returns data (often in JSON format).
  • Used by mobile apps, websites, and business systems.

2. HTTP/HTTPS protocol

Web APIs usually use HTTP or HTTPS:

  • HTTP – HyperText Transfer Protocol.
  • HTTPS – HTTP with encryption (secure).

3. Python requests library

In Python, the requests library is commonly used to call web APIs.

import requests response = requests.get("https://api.example.com/data") print(response.status_code) print(response.json())

4. HTTP GET method

GET is the HTTP method typically used to request data from an API.

# GET request example requests.get("https://api.exchangerate.host/latest")

5. Currency exchange API example

A business can use an API to get daily currency exchange rates. This allows automatic price updates in different currencies.

import requests url = "https://api.exchangerate.host/latest?base=USD" data = requests.get(url).json() uzs_rate = data["rates"]["UZS"] print("1 USD =", uzs_rate, "UZS")

6. APIs connect systems

APIs allow different systems to share data automatically:

  • CRM ⇄ ERP.
  • Website ⇄ payment gateway.
  • Mobile app ⇄ backend server.

7. JSON format

JSON = JavaScript Object Notation – a common data format for APIs.

{ "base": "USD", "rates": { "EUR": 0.92, "UZS": 12700.0 } }

8. Why APIs are important for business

Without APIs, many modern integrations would be impossible or very slow:

  • Online payments.
  • Real-time shipping tracking.
  • Automatic updates in e-commerce platforms.

Level 11 – Recap

  • REST API = web service for communication between programs.
  • APIs usually use HTTP/HTTPS.
  • Python requests is used for HTTP requests.
  • GET is used to retrieve data from an API.
  • Currency APIs help update prices automatically.
  • APIs connect CRM, ERP, websites and mobile apps.
  • JSON is a common data format for APIs.
  • Modern business integrations depend heavily on APIs.
