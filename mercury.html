Comments, print(), file extension, booleans and basic expressions.

1. Comments in Python

A comment starts with the # symbol. Everything after # on the same line is ignored by Python.

# This is a comment print("Hello") # this part is also a comment
  • Use comments to explain what the code does.
  • Comments do not change the result of the program.

2. Printing to the screen: print()

print() outputs text or values to the console.

print("Hello, world!") age = 18 print("Age:", age)
  • You can print strings, numbers, variables and expressions.
  • Use commas to print multiple values in one call.

3. Python file extension: .py

Python source files usually have the extension .py.

  • program.py
  • homework1.py

This tells the system and other people that this file contains Python code.

4. Booleans: True / False

A boolean value represents truth: True or False.

is_student = True is_admin = False print(5 > 3) # True print(2 == 10) # False

Booleans are used in if-statements and loops to control program flow.

5. Converting a string to int: int("5")

int("5") converts the string "5" to the integer 5.

text = "5" number = int(text) # 5 print(number + 3) # 8
  • Works only if the string looks like an integer number.
  • If the string is not numeric (e.g. "hello"), it raises ValueError.

6. Python is a high-level language

Python is called a high-level programming language because it is close to human language and hides low-level details like memory management.

  • Easier to read and write.
  • Good for beginners and for rapid development.

7. Indentation matters

In Python, indentation is part of the syntax. Blocks of code (inside if, for, functions, etc.) are defined by spaces.

if x > 0: print("Positive") # indented block print("Done") # outside the if

Wrong indentation can cause IndentationError or change program logic.

8. Expression: 3 + 4 * 2

Python follows normal math precedence: multiplication before addition.

value = 3 + 4 * 2 # 3 + 8 = 11 print(value) # 11

Use parentheses to change the order:

value = (3 + 4) * 2 # 7 * 2 = 14

Level 1 – Recap

  • Use # to start a comment.
  • print() shows values on the screen.
  • Python files end with .py.
  • Booleans: True and False.
  • int("5") returns integer 5.
  • Python is high-level and uses indentation as syntax.
  • 3 + 4 * 2 equals 11 (multiplication first).
