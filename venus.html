Lists, tuples, sets, dictionaries, strings, floats and immutability.

1. Lists in Python

A list is an ordered, changeable collection. It uses square brackets [].

numbers = [1, 2, 3] mixed = [1, "hello", True] print(numbers[0]) # 1 numbers.append(4) # [1, 2, 3, 4]

In the quiz, the correct list syntax is [1, 2, 3].

2. Immutable vs mutable

Mutable = can change after creation (e.g. list, dict, set). Immutable = cannot change (e.g. tuple, string, int).

t = (1, 2, 3) # tuple, immutable # t[0] = 10 # error lst = [1, 2, 3] # list, mutable lst[0] = 10 # [10, 2, 3]

3. Dictionaries: key–value pairs

A dictionary stores key–value pairs, uses {}.

user = { "name": "Anna", "age": 20 } print(user["name"]) # "Anna" user["age"] = 21 # change value

Dict is the correct answer for “type that stores key–value pairs”.

4. Float type

A float is a number with a decimal point, e.g. 3.14.

x = 3.14 print(type(x)) # <class 'float'>

5. Converting to string: str()

Use str() to convert a value to a string.

age = 20 text = "Age: " + str(age) print(text) # "Age: 20"

6. Strings are immutable

You cannot change a single character in a string in-place.

s = "hello" # s[0] = "H" # error s = "H" + s[1:] # "Hello"

7. Sets and duplicates

A set is an unordered collection of unique values.

s = {1, 2, 2, 3} print(s) # {1, 2, 3}

Sets automatically remove duplicates.

8. Tuple as dict key

Because tuples are immutable, they can be used as dictionary keys.

points = {} key = (10, 20) # tuple points[key] = "Player A" print(points[(10, 20)]) # "Player A"

Level 2 – Recap

  • List: [1, 2, 3], ordered and mutable.
  • Tuple: immutable, uses ().
  • Dict: key–value pairs, uses {}.
  • Float: decimal numbers like 3.14.
  • str() converts a value to string.
  • Strings and tuples are immutable.
  • Sets do not store duplicate values.
  • Tuples can be used as dictionary keys.
