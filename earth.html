Earth · Level 3 – Conditions & Loops

if-statements, for/while loops, range(), break, continue, nesting.

if for while break / continue
Back to Galaxy

1. if – starting a conditional statement

Use if to execute code only when a condition is True.

x = 10 if x > 5: print("x is greater than 5")

2. for-loop

Use for to iterate over a sequence (list, string, range, etc.).

for i in range(3): print(i) # 0, 1, 2

3. break – stop the loop

break immediately exits the closest loop.

for i in range(10): if i == 3: break print(i) # prints 0, 1, 2

4. range(3)

range(3) generates 0, 1, 2.

for i in range(3): print(i)

5. continue – skip to next iteration

continue skips the rest of the current iteration and moves to the next one.

for i in range(5): if i == 2: continue print(i) # prints 0, 1, 3, 4

6. while loop can run forever

A while loop repeats while its condition is True. If the condition never becomes False, the loop can run forever.

while True: print("infinite loop") # be careful!
