Comparison operators, loops, data structures, ML libraries, CRM and KPIs.

1. Comparison operator ==

Use == to compare two values (equality).

print(2 == 2) # True print(2 == 3) # False

= is assignment, == is comparison.

2. while loop keyword

A while loop repeats as long as its condition is True.

i = 0 while i < 3: print(i) i += 1

3. Not a Python built-in data structure

Built-in data structures: list, set, dict, tuple. arraylist is not a Python built-in (it is a Java structure).

4. Machine learning library: scikit-learn

scikit-learn is a popular ML library for Python.

from sklearn.linear_model import LinearRegression model = LinearRegression() # model.fit(X, y)

5. CRM system

CRM = Customer Relationship Management. It is used to manage customers and sales.

  • Store client contacts.
  • Track deals and opportunities.
  • Plan follow-ups.

6. Nested lists

A list can contain another list inside it.

matrix = [ [1, 2, 3], [4, 5, 6] ]

7. Integer division // in Python

// is integer (floor) division.

print(10 // 3) # 3 print(10 / 3) # 3.3333...

8. if without else

You can write if without else. else is optional.

if x > 0: print("Positive")

Level 8 – Recap

  • Use == to compare values.
  • while starts a while loop.
  • arraylist is not a Python built-in structure.
  • scikit-learn is used for machine learning in Python.
  • CRM manages customers and sales.
  • Lists can contain other lists.
  • 10 // 3 equals 3.
  • if does not always require else.
