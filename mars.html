Mars · Level 4 – Functions & Modules

def, calling functions, return, imports and using modules like math.

def return import modules
Back to Galaxy

1. Defining a function: def

Use def to define a new function.

def greet(): print("Hello!") def add(a, b): return a + b

2. Calling a function

To call a function named greet, use parentheses: greet().

greet() # function call result = add(2, 3) print(result) # 5

3. Returning values: return

return sends a value back to the caller and ends the function.

def square(x): return x * x print(square(4)) # 16

4. Where to put imports

Usually we place import statements at the top of the file.

import math import random # rest of the code...

This makes it easy to see which modules are used in the script.

5. What does "import math" do?

import math loads the math module with functions like sqrt, sin, etc.

import math print(math.pi) # 3.14159... print(math.sqrt(16)) # 4.0

6. Parameters and multiple arguments

A function can have zero, one or many parameters.

def welcome(name, city): print("Hello", name, "from", city) welcome("Anna", "Tashkent")

7. Functions without return

If a function has no return statement, it returns None by default.

def do_nothing(): pass result = do_nothing() print(result) # None

8. Importing your own modules

You can import your own Python files as modules (if they are in the same folder or package).

# file: helpers.py def add_one(x): return x + 1 # file: main.py import helpers print(helpers.add_one(10)) # 11

Level 4 – Recap

  • Use def to define a function.
  • Call a function with parentheses: greet().
  • return sends a value back and ends the function.
  • Imports are usually at the top of the file.
  • import math loads the math module.
  • Functions can have many parameters.
  • No return ⇒ function returns None.
  • You can import your own Python files as modules.
